Project Overview

This project is a static website consisting of four main pages:

Home (index.html)
Services
Case Studies
Contact

The website is designed to simulate a modern digital agency landing experience, with attention to typography, layout, and responsiveness.


static-web/

├── index.html

├── pages/

│   ├── case-studies.html

├── css/

├── js/

│   └── app.js

└── images/

structure, responsive design, and interactive UI without using any frameworks.

Features
 Fully responsive design
 Clean UI with modern typography
 Modular CSS (global + page-specific)
 Mobile hamburger navigation
 Scroll reveal animations (IntersectionObserver)
 Services accordion section
 Contact form with validation

Tech Stack
HTML5
CSS3 (Flexbox + Grid)
JavaScript (Vanilla JS)
 
Highlights
Used CSS variables for consistent design
Applied Grid for layouts & Flexbox for alignment
Single JS file handling all interactivity
Optimized structure for scalability and readability
